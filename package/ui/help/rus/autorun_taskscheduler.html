Планировщик задач

При определенных обстоятельствах имеет смысл запустить задачу, определенную в планировщике задач, с помощью автозапуска.

Команда оболочки для запуска такой задачи, например, /usr/syno/bin/synoschedtask --run id=14 check_time=0. "Task-ID" (14 в примере) может быть получен командой synoschedtask --get. При определенных обстоятельствах также имеет смысл выполнить команду, указанную как "Command", напрямую, минуя планировщик задач.

Команда запуска планировщика выполняется мгновенно, и задача запускается как независимый процесс. Если вы хотите использовать управление светодиодом для отображения запущенной задачи или извлечения диска после завершения задачи, то в вашем скрипте после запуска задачи необходимо дождаться окончания этого процесса с помощью цикла, как показано ниже:

sleep 60
while [ "$(/bin/pidof <processName>)" ]; do
sleep 60
done
