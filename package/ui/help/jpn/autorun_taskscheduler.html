タスクスケジューラ

タスクスケジューラーで定義されたタスクをオートランで起動することが有効な場合があります。

タスクID（例では14）は、synoschedtask --getで求めることができます。状況によっては、「command」で指定されたコマンドを直接実行し、タスクスケジューラーをバイパスすることも有効である。

すると、スケジューラのスタートコマンドはあっという間に終了し、タスクは独立したプロセスとして実行されます。LED制御で実行中のタスクを表示したり、タスク終了後にハードディスクをイジェクトしたりしたい場合は、スクリプト内でタスク開始後に次のようなループでプロセスの終了を待つ必要があります：

sleep 60
while [ "$(/bin/pidof<プロセス名>)" ]; do
sleep 60
done
