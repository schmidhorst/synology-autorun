Programador de tareas

En determinadas circunstancias es útil iniciar una tarea definida en el programador de tareas mediante ejecución automática.

El comando shell para iniciar una tarea de este tipo es, por ejemplo, /usr/syno/bin/synoschedtask --run id=14 check_time=0. El "ID de tarea" (14 en el ejemplo) puede determinarse con synoschedtask --get. En determinadas circunstancias, también es útil ejecutar directamente el comando especificado como "comando" y omitir el programador de tareas.

De este modo, el comando de inicio del programador se ejecuta en un abrir y cerrar de ojos y la tarea se ejecuta como un proceso independiente. Si desea utilizar el control LED para indicar la tarea en ejecución o para expulsar el disco duro después de que la tarea haya finalizado, debe esperar en su script el final del proceso después del inicio de la tarea con un bucle como este:

sleep 60
while [ "$(/bin/pidof <nombre del proceso>)" ]; do
sleep 60
done
