Planificateur de tâches

Dans certaines circonstances, il est utile de lancer une tâche définie dans le planificateur de tâches via l'exécution automatique.

La commande shell pour lancer une telle tâche est par exemple /usr/syno/bin/synoschedtask --run id=14 check_time=0. L'"ID de la tâche" (14 dans l'exemple) peut être déterminé avec synoschedtask --get. Dans certaines circonstances, il est également utile d'exécuter directement la commande spécifiée comme "command" et de contourner le planificateur de tâches.

La commande de démarrage du planificateur est alors exécutée en un rien de temps et la tâche s'exécute comme un processus indépendant. Si vous voulez utiliser la commande LED pour indiquer la tâche en cours ou pour éjecter le disque dur une fois la tâche terminée, vous devez attendre dans votre script la fin du processus après le démarrage de la tâche avec une boucle comme celle-ci :

sleep 60
while [ "$(/bin/pidof <nom du processus>)] ; do
sleep 60
done
