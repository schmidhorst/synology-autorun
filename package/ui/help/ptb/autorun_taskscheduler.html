Agendador de tarefas

Sob certas circunstâncias, é útil iniciar uma tarefa definida no agendador de tarefas via autorun.

O comando shell para iniciar tal tarefa é, por exemplo, /usr/syno/bin/synoschedtask --run id=14 check_time=0. A "identificação da tarefa" (14 no exemplo) pode ser determinada com synoschedtask --get. Sob certas circunstâncias também é útil executar o comando especificado como "comando" diretamente e contornar o agendador de tarefas.

O comando de partida do programador é então feito em pouco tempo e a tarefa é executada como um processo independente. Se você quiser usar o controle LED para indicar a tarefa em execução ou para ejetar o disco rígido após a tarefa ter terminado, você deve esperar em seu script o final do processo após o início da tarefa com um loop como este:

sleep 60
enquanto ["$(/bin/pidof <processar nome>)" ]; do
sleep 60
feito
