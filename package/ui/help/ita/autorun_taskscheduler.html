Pianificazione attività

In alcune circostanze è utile avviare un'attività definita nel pianificatore attività tramite l'esecuzione automatica.

Il comando di shell per avviare un task di questo tipo è, ad esempio, /usr/syno/bin/synoschedtask --run id=14 check_time=0. L'"ID del task" (14 nell'esempio) può essere determinato con synoschedtask --get. In alcune circostanze è anche utile eseguire direttamente il comando specificato come "command" e bypassare il task scheduler.

Il comando di avvio dello schedulatore viene eseguito in un attimo e il task viene eseguito come processo indipendente. Se si desidera utilizzare il controllo LED per indicare l'esecuzione del task o per espellere il disco rigido al termine del task, è necessario attendere nel proprio script la fine del processo dopo l'avvio del task con un ciclo come questo:

sleep 60
while [ "$(/bin/pidof <nome processo>)" ]; do
sleep 60
done
