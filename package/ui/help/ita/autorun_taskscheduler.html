Pianificazione attività

In alcune circostanze è opportuno avviare un task definito nel task scheduler tramite autorun.

Il comando di shell per avviare un task di questo tipo è, ad esempio, /usr/syno/bin/synoschedtask --run id=14 check_time=0. Il "Task-ID" (14 nell'esempio) può essere ottenuto con synoschedtask --get. In alcune circostanze ha senso anche eseguire direttamente il comando specificato come "Command" e bypassare il task scheduler.

Il comando di avvio dello schedulatore viene eseguito in un attimo e il task viene eseguito come processo indipendente. Se si desidera utilizzare il controllo LED per visualizzare l'attività in corso o per espellere il disco al termine dell'attività, è necessario attendere la fine del processo nel proprio script dopo l'avvio dell'attività con un ciclo come questo:

sleep 60
while [ "$(/bin/pidof <nomeprocesso>)" ]; do
sleep 60
done
