Планировщик задач

При определенных обстоятельствах полезно запустить задачу, определенную в планировщике задач, через автозапуск.

Команда оболочки для запуска такой задачи - это, например, /usr/syno/bin/synoschedtask --run id=14 check_time=0. "ID задачи" (14 в примере) можно определить с помощью synoschedtask --get. При определенных обстоятельствах полезно также выполнить команду, указанную как "command", напрямую, минуя планировщик задач.

В этом случае команда запуска планировщика выполняется мгновенно, а задача запускается как независимый процесс. Если вы хотите использовать управление светодиодом для индикации запущенной задачи или для извлечения жесткого диска после завершения задачи, вы должны в своем сценарии дождаться окончания процесса после запуска задачи с помощью цикла, подобного следующему:

sleep 60
while [ "$(/bin/pidof <имя процесса>)" ]; do
sleep 60
done
