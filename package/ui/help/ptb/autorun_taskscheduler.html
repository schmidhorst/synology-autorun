Agendador de tarefas

Sob certas circunstâncias, faz sentido iniciar uma tarefa definida no agendador de tarefas pela autorun.

O comando shell para iniciar tal tarefa é, por exemplo, /usr/syno/bin/synoschedtask --run id=14 check_time=0. O "Task-ID" (14 no exemplo) pode ser obtido por synoschedtask --get. Sob certas circunstâncias, também faz sentido executar o comando especificado como "Comando" diretamente e contornar o agendador de tarefas.

O comando de início do agendador é feito em pouco tempo e a tarefa é executada como um processo independente. Se você quiser usar o controle LED para exibir a tarefa em execução ou para ejetar o disco após a tarefa ser concluída, você tem que esperar em seu script após o início da tarefa com um loop como este para o final desse processo:

sleep 60
enquanto ["$(/bin/pidof <processName>) " ]; do
sleep 60
feito
