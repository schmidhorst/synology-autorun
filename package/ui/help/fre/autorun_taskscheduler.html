Planificateur de tâches

Dans certaines circonstances, il est judicieux de lancer une tâche définie dans le planificateur de tâches par exécution automatique.

La commande shell pour lancer une telle tâche est par exemple /usr/syno/bin/synoschedtask --run id=14 check_time=0. Le "Task-ID" (14 dans l'exemple) peut être obtenu par synoschedtask --get. Dans certaines circonstances, il est également judicieux d'exécuter directement la commande spécifiée en tant que "Commande" et de contourner le planificateur de tâches.

La commande de démarrage du planificateur est exécutée en un rien de temps et la tâche s'exécute comme un processus indépendant. Si vous souhaitez utiliser la commande LED pour afficher la tâche en cours ou pour éjecter le disque une fois la tâche terminée, vous devez attendre la fin du processus dans votre script après le démarrage de la tâche, au moyen d'une boucle de ce type :

sleep 60
while [ "$(/bin/pidof <nomduprocessus>)] ; do
sleep 60
done
