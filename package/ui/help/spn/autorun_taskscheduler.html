Programador de tareas

En determinadas circunstancias, tiene sentido iniciar una tarea definida en el programador de tareas mediante ejecución automática.

El comando shell para iniciar una tarea de este tipo es, por ejemplo, /usr/syno/bin/synoschedtask --run id=14 check_time=0. El "Task-ID" (14 en el ejemplo) puede obtenerse mediante synoschedtask --get. En determinadas circunstancias también tiene sentido ejecutar directamente el comando especificado como "Comando" y saltarse el programador de tareas.

El comando de inicio del programador se realiza en un abrir y cerrar de ojos y la tarea se ejecuta como un proceso independiente. Si desea utilizar el control LED para mostrar la tarea en ejecución o para expulsar el disco después de que la tarea haya terminado, tiene que esperar en su script después del inicio de la tarea con un bucle como este para el final de ese proceso:

sleep 60
while [ "$(/bin/pidof <nombredelproceso>)" ]; do
sleep 60
done
