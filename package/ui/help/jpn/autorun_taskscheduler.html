タスクスケジューラ

ある状況下では、タスクスケジューラーで定義されたタスクをオートランで起動することが理にかなっています。

タスクID（例では14）は、synoschedtask --getで取得することができます。 ある状況下では、「コマンド」として指定されたコマンドを直接実行し、タスクスケジューラを迂回することも意味がある。

スケジューラーのスタートコマンドはあっという間に終了し、タスクは独立したプロセスとして実行されます。 LEDコントロールを使って実行中のタスクを表示したり、タスク終了後にディスクをイジェクトしたりしたい場合は、タスク開始後にスクリプト内で次のようなループでそのプロセスの終了を待つ必要があります：

sleep 60
while [ "$(/bin/pidof<processName>)" ]; do
sleep 60
done
