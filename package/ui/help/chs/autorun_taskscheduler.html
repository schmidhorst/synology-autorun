任务调度器

在某些情况下，通过自动运行来启动一个在任务调度器中定义的任务是有意义的。

启动这样一个任务的shell命令是/usr/syno/bin/synoschedtask --run id=14 check_time=0."任务ID"（在这个例子中是14）可以通过synoschedtask --get获得。 在某些情况下，直接执行指定为 "Command "的命令并绕过任务调度器也是有意义的。

调度器的启动命令在短时间内完成，任务作为一个独立的进程运行。 如果你想用LED控制显示正在运行的任务或者在任务结束后弹出磁盘，你必须在任务开始后用这样的循环等待该进程的结束：

sleep 60
while [ "$(/bin/pidof<processName>)" ]; do
sleep 60
done
