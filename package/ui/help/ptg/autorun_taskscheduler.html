Agendador de Tarefas

Em determinadas circunstâncias, faz sentido iniciar uma tarefa definida no agendador de tarefas pela autorun.

O comando shell para iniciar uma tal tarefa é por exemplo /usr/syno/bin/synoschedtask --run id=14 check_time=0. O "Task-ID" (14 no exemplo) pode ser obtido por synoschedtask --get. Em determinadas circunstâncias também faz sentido executar directamente o comando especificado como "Comando" e contornar o programador de tarefas.

O comando de início do agendador é feito em pouco tempo e a tarefa é executada como um processo independente. Se quiser usar o controlo LED para exibir a tarefa em execução ou para ejectar o disco após a tarefa estar concluída, tem de esperar no seu script após o início da tarefa com um loop como este para o fim desse processo:

dormir 60
enquanto ["$(/bin/pidof <processName>) " ]; fazer
dormir 60
feito
